ü©∫ Heart Rate Values

What heart rates to expect:
  • Resting (sitting): 60-100 BPM
  • Light activity: 100-120 BPM
  • Moderate activity: 120-150 BPM
  • Vigorous activity: 150+ BPM

If you get unusual readings:

  • Very low (< 40 BPM): Check H10 contact with skin
  • Very high (> 200 BPM): Sensor might be picking up movement, not heart rate
  • Erratic readings: Wet the electrodes or adjust strap tightness

üÜò Troubleshooting

If nothing happens when you click Test Connection:

  • Check that you're using Chrome, Edge, or Opera (not Safari)
  • Make sure JavaScript is enabled
  • Check browser console for error messages
  • Try refreshing the page

If device picker doesn't show your H10:

  • Make sure H10 is turned on (LED should be blinking)
  • Check that it's not connected to Polar Beat or other apps
  • Try turning H10 off and on again
  • Move H10 closer to your computer

If connection keeps failing:

  • Reset your H10 (hold button for 10 seconds if available)
  • Clear browser cache and reload page
  • Check Mac Bluetooth is enabled in System Preferences
  • Try a different browser
